## 1. 感知机

### 1.1 什么是感知机？

感知机是最简单的神经网络：

```
输入 → 权重 → 求和 → 激活 → 输出

x1 ──w1──┐
          │
x2 ──w2──┼──→ (Σ + b) → activation → y
          │
x3 ──w3──┘
```

### 1.2 代码实现

```python
import numpy as np

class Perceptron:
    def __init__(self, input_dim):
        # 初始化权重
        self.weights = np.random.randn(input_dim) * 0.1
        self.bias = 0

    def forward(self, x):
        # 前向传播
        z = np.dot(x, self.weights) + self.bias
        return self.activation(z)

    def activation(self, x):
        # 阶跃函数
        return np.where(x > 0, 1, 0)

# 使用
p = Perceptron(3)
output = p.forward([1, 2, 3])
print(output)  # 1 或 0
```

## 2. 前向传播

### 2.1 概念

前向传播：输入 → 隐藏层 → 输出

```python
def forward(x, W1, b1, W2, b2):
    # 隐藏层
    z1 = np.dot(x, W1) + b1
    a1 = np.tanh(z1)  # 激活函数

    # 输出层
    z2 = np.dot(a1, W2) + b2
    a2 = sigmoid(z2)

    return a2
```

## 3. 损失函数

### 3.1 均方误差 (MSE)

用于回归任务：

```
L = (1/n) * Σ(y - ŷ)²
```

```python
def mse_loss(y_pred, y_true):
    return np.mean((y_pred - y_true) ** 2)
```

### 3.2 交叉熵 (Cross Entropy)

用于分类任务：

```
L = -Σ y * log(ŷ)
```

```python
def cross_entropy(y_pred, y_true):
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred))
```

## 4. 反向传播

### 4.1 链式法则

```
∂L/∂w = ∂L/∂a * ∂a/∂z * ∂z/∂w
```

### 4.2 示例

```python
# 简化版反向传播
def backward(x, y, y_pred, W1, W2):
    # 输出层梯度
    dL_dy = y_pred - y
    dL_dW2 = np.dot(a1.T, dL_dy)

    # 隐藏层梯度
    dL_da1 = np.dot(dL_dy, W2.T)
    dL_dz1 = dL_da1 * (1 - a1**2)  # tanh 导数
    dL_dW1 = np.dot(x.T, dL_dz1)

    # 更新权重
    W1 -= learning_rate * dL_dW1
    W2 -= learning_rate * dL_dW2
```

## 5. 梯度下降

### 5.1 批量梯度下降

```python
def train(X, y, lr=0.01, epochs=100):
    # 初始化权重
    W = np.random.randn(X.shape[1]) * 0.1

    for epoch in range(epochs):
        # 计算预测
        y_pred = X @ W

        # 计算梯度
        gradient = X.T @ (y_pred - y) / len(X)

        # 更新权重
        W -= lr * gradient

        # 打印损失
        loss = np.mean((y_pred - y) ** 2)
        if epoch % 10 == 0:
            print(f"Epoch {epoch}: loss {loss:.4f}")

    return W
```
