## 1.1 梯度下降

### 1.1.1 什么是梯度下降？

**梯度**：函数在某点上升最快的方向
**下降**：往梯度的反方向走

```
w = w - learning_rate * gradient
```

### 1.1.2 直观理解

想象你在山顶，需要走到山谷：

```
山顶 (高损失)
   ↓ 沿着最陡的方向往下走
   ↓
山谷 (最低损失)
```

## 1.2 链式法则

### 1.2.1 复合函数求导

```
y = f(g(x))

dy/dx = dy/dg * dg/dx
```

### 1.2.2 示例

```
f(x) = sin(x²)

f'(x) = cos(x²) * 2x
       = 2x * cos(x²)
```

## 1.3 反向传播

### 1.3.1 前向传播

```
输入 x → 隐藏层 h = f(Wx+b) → 输出 y
```

### 1.3.2 反向传播

```
误差从后往前传：
∂L/∂y → ∂L/∂h → ∂L/∂W → ∂L/∂b
```

### 1.3.3 计算流程

```
1. 前向计算每个节点的输出
2. 计算输出层误差
3. 从后往前计算每个参数的梯度
4. 更新参数
```

## 1.4 代码实现

### 1.4.1 简单神经网络

```python
import numpy as np

class SimpleNN:
    def __init__(self):
        # 初始化权重
        self.W1 = np.random.randn(2, 4) * 0.1
        self.b1 = np.zeros((1, 4))
        self.W2 = np.random.randn(4, 1) * 0.1
        self.b2 = np.zeros((1, 1))

    def sigmoid(self, x):
        return 1 / (1 + np.exp(-x))

    def sigmoid_derivative(self, x):
        return x * (1 - x)

    def forward(self, X):
        # 隐藏层
        self.z1 = X @ self.W1 + self.b1
        self.a1 = self.sigmoid(self.z1)
        # 输出层
        self.z2 = self.a1 @ self.W2 + self.b2
        self.a2 = self.sigmoid(self.z2)
        return self.a2

    def backward(self, X, y, learning_rate=0.1):
        m = X.shape[0]

        # 输出层误差
        delta2 = (self.a2 - y) * self.sigmoid_derivative(self.a2)

        # 隐藏层误差
        delta1 = (delta2 @ self.W2.T) * self.sigmoid_derivative(self.a1)

        # 更新权重
        self.W2 -= learning_rate * (self.a1.T @ delta2) / m
        self.b2 -= learning_rate * np.sum(delta2, axis=0) / m
        self.W1 -= learning_rate * (X.T @ delta1) / m
        self.b1 -= learning_rate * np.sum(delta1, axis=0) / m

    def train(self, X, y, epochs=10000):
        for i in range(epochs):
            output = self.forward(X)
            self.backward(X, y)
            if i % 1000 == 0:
                loss = np.mean((output - y) ** 2)
                print(f"Epoch {i}, Loss: {loss:.4f}")

# 测试
nn = SimpleNN()
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])  # XOR
nn.train(X, y)
print(nn.forward(X))
```
