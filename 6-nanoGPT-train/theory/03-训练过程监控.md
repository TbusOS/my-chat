## 1. Loss 曲线解读

### 1.1 典型训练输出

```
step    0: train loss 4.2241, val loss 4.2306   ← 随机初始化
step  500: train loss 1.8135, val loss 1.9912   ← 快速下降
step 1000: train loss 1.5623, val loss 1.7104   ← 下降变缓
step 2500: train loss 1.2018, val loss 1.4856   ← 间隙变大
step 5000: train loss 0.9214, val loss 1.4701   ← 趋于收敛
```

### 1.2 状态判断

| 模式 | train loss | val loss | 诊断 | 对策 |
|------|-----------|----------|------|------|
| 正常 | 下降 | 下降 | 继续训练 | 无需调整 |
| 过拟合 | 下降 | 上升 | 记住了训练数据 | 增加 dropout / 数据 |
| 欠拟合 | 停滞 | 停滞 | 容量不足 | 增大模型 / 学习率 |
| 发散 | 震荡/NaN | 震荡/NaN | 学习率过大 | 降低学习率 |
| 数据问题 | 极高不降 | 极高不降 | 数据损坏 | 重新 prepare.py |

## 2. 学习率调度：Cosine Annealing + Warmup

```python
def get_lr(it, warmup_iters, lr_decay_iters, learning_rate, min_lr):
    """nanoGPT 的学习率调度（简化版）"""
    import math
    if it < warmup_iters:                          # Warmup：线性增长
        return learning_rate * it / warmup_iters
    if it > lr_decay_iters:                        # 衰减结束：最小值
        return min_lr
    decay_ratio = (it - warmup_iters) / (lr_decay_iters - warmup_iters)
    coeff = 0.5 * (1.0 + math.cos(math.pi * decay_ratio))  # 余弦衰减
    return min_lr + coeff * (learning_rate - min_lr)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `learning_rate` | 6e-4 | 峰值学习率 |
| `min_lr` | 6e-5 | 最小学习率（峰值的 1/10） |
| `warmup_iters` | 100 | 预热步数 |
| `lr_decay_iters` | 5000 | 衰减总步数 |

## 3. 过拟合与欠拟合

### 过拟合解决方案

```bash
python train.py --dropout=0.2                          # 增加 dropout
python train.py --n_layer=4 --n_head=4 --n_embd=256    # 减小模型
python train.py --max_iters=2000                       # 提前终止
```

### 欠拟合解决方案

```bash
python train.py --n_layer=8 --n_head=8 --n_embd=512    # 增大模型
python train.py --learning_rate=1e-3                    # 增大学习率
python train.py --max_iters=20000                       # 增加训练步数
```

| wandb 记录指标 | 说明 |
|----------------|------|

不使用 wandb 时，可用简易日志替代：

```python
"""simple_logger.py - 简易训练日志"""
import json, matplotlib.pyplot as plt

class TrainLogger:
    def __init__(self, path="train_log.json"):
        self.path = path
        self.records = []

    def log(self, step, train_loss, val_loss=None, lr=None):
        self.records.append({"step": step, "train_loss": train_loss,
                             "val_loss": val_loss, "lr": lr})

    def save(self):
        with open(self.path, 'w') as f:
            json.dump(self.records, f, indent=2)

    def plot(self):
        steps = [r["step"] for r in self.records]
        plt.plot(steps, [r["train_loss"] for r in self.records], label='Train')
        plt.plot([r["step"] for r in self.records if r["val_loss"]],
                 [r["val_loss"] for r in self.records if r["val_loss"]], label='Val')
        plt.xlabel('Step'); plt.ylabel('Loss'); plt.legend(); plt.grid(True)
        plt.savefig('loss_curve.png', dpi=150); plt.close()
```

## 6. nanoGPT 核心配置参数

### 模型参数

| 参数 | 默认值 | 说明 | 调优建议 |
|------|--------|------|----------|
| `n_layer` | 12 | Transformer 层数 | 入门 4-6，充足资源 8-12 |
| `n_head` | 12 | 注意力头数 | 通常等于 n_layer |
| `n_embd` | 768 | embedding 维度 | n_head 的 64 倍 |
| `block_size` | 1024 | 上下文窗口 | 入门 256，充足 512-1024 |
| `dropout` | 0.0 | Dropout 比率 | 过拟合时 0.1-0.2 |

### 训练参数

| 参数 | 默认值 | 说明 | 调优建议 |
|------|--------|------|----------|
| `batch_size` | 12 | 每步样本数 | 越大越好（受显存限制） |
| `max_iters` | 600000 | 最大训练步数 | 小数据集 3000-10000 |
| `grad_clip` | 1.0 | 梯度裁剪阈值 | 防止梯度爆炸 |
| `gradient_accumulation_steps` | 1 | 梯度累积 | 显存不足时增大 |
| `eval_interval` | 2000 | 评估间隔 | 调试时可减小 |
