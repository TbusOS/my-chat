## 2. 数据清洗

```python
import re

def clean_text(raw_text: str) -> str:
    """基础数据清洗"""
    text = re.sub(r'<[^>]+>', '', raw_text)           # 去除 HTML 标签
    text = text.replace('\r\n', '\n')                  # 统一换行符
    text = re.sub(r'\n{3,}', '\n\n', text)             # 去除多余空行
    lines = [line.strip() for line in text.split('\n')]
    return '\n'.join(lines).strip()

def is_quality_text(text: str, min_length: int = 50) -> bool:
    """质量过滤：去除过短或高重复文本"""
    if len(text) < min_length:
        return False
    words = text.split()
    if not words:
        return False
    max_freq = max(words.count(w) for w in set(words))
    return max_freq / len(words) <= 0.3
```

## 3. Tokenization：字符级 vs BPE

### 3.1 字符级

```python
chars = sorted(list(set(text)))
stoi = {ch: i for i, ch in enumerate(chars)}  # 字符 → 整数
itos = {i: ch for i, ch in enumerate(chars)}  # 整数 → 字符
encode = lambda s: [stoi[c] for c in s]
decode = lambda l: ''.join([itos[i] for i in l])
```

### 3.2 BPE（tiktoken）

```python
import tiktoken
enc = tiktoken.get_encoding("gpt2")
tokens = enc.encode("Hello, world!")  # [15496, 11, 995, 0]
text = enc.decode(tokens)             # "Hello, world!"
```

### 3.3 对比与选择

| 维度 | 字符级 | BPE |
|------|--------|-----|
| 词表大小 | ~65 | ~50257 |
| 序列长度 | 长（每字符一个 token） | 短（一个词一个 token） |
| 适用场景 | 入门学习、小语料 | 工业界、复现 GPT-2 |
| 中文支持 | 直接可用 | gpt2 编码效率低（一个汉字 2-3 token） |

## 5. prepare.py 解析

```python
"""shakespeare_char/prepare.py 简化版"""
import os, pickle, requests
import numpy as np

# 下载数据
url = 'https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt'
data = requests.get(url).text

# 构建词表
chars = sorted(list(set(data)))
stoi = {ch: i for i, ch in enumerate(chars)}
encode = lambda s: [stoi[c] for c in s]

# 划分 90% 训练 / 10% 验证
n = len(data)
train_ids = np.array(encode(data[:int(n*0.9)]), dtype=np.uint16)
val_ids = np.array(encode(data[int(n*0.9):]), dtype=np.uint16)

# 保存
train_ids.tofile('train.bin')
val_ids.tofile('val.bin')
pickle.dump({'vocab_size': len(chars), 'itos': {i:c for i,c in enumerate(chars)}, 'stoi': stoi},
            open('meta.pkl', 'wb'))
```

## 6. 实战：准备自定义中文数据集

```python
"""prepare_chinese.py - 中文字符级数据集"""
import os, pickle
import numpy as np

with open('my_chinese_text.txt', 'r', encoding='utf-8') as f:
    data = f.read()

# 清洗
data = data.replace('\r\n', '\n')
data = '\n'.join(line.strip() for line in data.split('\n'))

# 构建词表
chars = sorted(list(set(data)))
stoi = {ch: i for i, ch in enumerate(chars)}
itos = {i: ch for i, ch in enumerate(chars)}
encode = lambda s: [stoi[c] for c in s]

# 划分并保存
n = len(data)
os.makedirs('data/chinese_char', exist_ok=True)
np.array(encode(data[:int(n*0.9)]), dtype=np.uint16).tofile('data/chinese_char/train.bin')
np.array(encode(data[int(n*0.9):]), dtype=np.uint16).tofile('data/chinese_char/val.bin')
pickle.dump({'vocab_size': len(chars), 'itos': itos, 'stoi': stoi},
            open('data/chinese_char/meta.pkl', 'wb'))
print(f"词表: {len(chars)} | 训练: {int(n*0.9):,} chars | 验证: {n-int(n*0.9):,} chars")
```

```bash
# 训练自定义数据
python train.py \
    --dataset=chinese_char \
    --n_layer=6 --n_head=6 --n_embd=384 \
    --block_size=256 --batch_size=64 \
    --max_iters=5000 \
    --device=mps --compile=False \
    --out_dir=out-chinese
```

## 7. 注意事项

| 问题 | 说明 |
|------|------|
| 数据量太小 | 至少 100KB，推荐 1MB+ |
| 中文 BPE 效率 | tiktoken gpt2 编码对中文效率低 |
| 数据泄漏 | 确保验证集不包含训练集内容 |
| 编码类型 | vocab_size < 65536 用 uint16，否则 uint32 |
