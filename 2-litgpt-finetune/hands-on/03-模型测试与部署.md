## 1.1 评估方法

### 1.1.1 自动评估

```python
# 使用 litgpt 评估
litgpt evaluate \
  --model out/lora-phi2/final \
  --data alpaca \
  --tasks "arc_easy,arc_challenge"
```

### 1.1.2 人工评估

```python
from litgpt import LLM

llm = LLM.load(
    name="microsoft/phi-2",
    finetune_path="out/lora-phi2/final"
)

# 测试用例
test_cases = [
    "解释什么是机器学习",
    "写一个快速排序函数",
    "翻译: Hello World"
]

for prompt in test_cases:
    response = llm.chat(
        messages=[{"role": "user", "content": prompt}]
    )
    print(f"Q: {prompt}")
    print(f"A: {response}\n")
```

## 1.4 API 服务

### 1.4.1 使用 Ollama

```bash
# 创建 Modelfile
echo "
FROM microsoft/phi-2
ADAPTER ./out/lora-phi2/final
" > Modelfile

# 导入 Ollama
ollama create my-model -f Modelfile

# 使用
ollama run my-model "你好"
```

### 1.4.2 使用 FastAPI

```python
from fastapi import FastAPI
from litgpt import LLM
import uvicorn

app = FastAPI()

llm = None

@app.on_event("startup")
def load_model():
    global llm
    llm = LLM.load(
        name="microsoft/phi-2",
        finetune_path="out/lora-phi2/final"
    )

@app.post("/chat")
def chat(message: dict):
    response = llm.chat(
        messages=[{"role": "user", "content": message["content"]}]
    )
    return {"response": response}

if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

启动服务：

```bash
uvicorn server:app --host 0.0.0.0 --port 8000
```

调用：

```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"content": "你好"}'
```

## 1.5 Docker 部署

### 1.5.1 Dockerfile

```dockerfile
FROM python:3.10

WORKDIR /app

RUN pip install litgpt fastapi uvicorn

COPY out/lora-phi2 /model

COPY server.py /app/

EXPOSE 8000

CMD ["uvicorn", "server:app", "--host", "0.0.0.0"]
```

### 1.5.2 构建运行

```bash
docker build -t my-llm .
docker run -p 8000:8000 my-llm
```
