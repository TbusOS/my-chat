关键洞察：AI 最独特的地方在于——它的「程序间通信」用的是 JSON + 自然语言的混合体。 传统软件之间只能用严格的接口协议（必须一个字段都不能错），而 AI Agent 之间可以用「半结构化」的方式沟通： 外面套个 JSON 格式让机器能解析，里面填的是人话让 AI 能理解语义。

这就像你填表格——表格的格式是固定的（JSON），但「备注」栏你可以自由发挥（自然语言）。

输入一句话，看 AI 是怎么「看到」你说的文字的

1 你看到的（自然语言）
2 AI 看到的（Token 序列）
3 AI 内心的（向量 / 嵌入）
为什么是 Token 而不是「字」？
• 英文 "unhappiness" 会被拆成 ["un", "happiness"]，因为 "un-" 是一个有意义的前缀
• 中文 "今天天气" 可能被拆成 ["今天", "天气"] 或 ["今", "天", "天", "气"]，取决于模型的词表
• Token 是 AI 的「最小理解单位」，就像人类大脑处理语言时用的也不是单个笔画，而是词组和概念
• API 收费按 Token 数量计算，所以中文比英文贵（同样的意思，中文 Token 更多）

👤 你输入中文 ⟶ 📦 JSON 请求 ⟶ 🔬 拆成 Token ⟶ 🔢 变成向量 ⟶ 🧠 神经网络计算 ⟶ 🔬 输出 Token ⟶ 💬 拼成自然语言
