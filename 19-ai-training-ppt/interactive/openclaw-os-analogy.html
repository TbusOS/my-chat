OpenClaw = AI 的智能手机

用你每天都在用的手机，理解 AI Agent 框架的五大核心

想象一下：大模型 (GPT/Claude) 是「App」，OpenClaw 是运行它的「手机操作系统」

没有操作系统，App 就不知道怎么接收消息、存储数据、调用摄像头 —— AI 也一样

🔔 Gateway = 手机的通知中心

所有消息的统一入口，决定谁来处理、怎么排队

📱 手机上

💬 微信、钉钉、短信的消息都从通知中心进来
🔔 通知中心决定弹窗、横幅还是静默
👥 多个 App 同时来消息？排队处理

✿ OpenClaw 里

📨 飞书、Slack、API 的请求都从 Gateway 进来
📋 Gateway 路由到正确的 Agent 处理
🔧 Lane Queue 机制，多任务排队不乱

🧠 Brain = 手机的处理器 (CPU)

真正「思考」的核心，决定做什么、怎么做

📱 手机上

⚡ CPU 运行所有 App 的逻辑
🔄 多任务切换，一会儿刷微博一会儿回消息
🤖 A17/骁龙芯片决定手机有多聪明

✿ OpenClaw 里

💡 Brain 调用大模型 (GPT/Claude) 来「思考」
🔁 Agentic Loop: 思考 ⟶ 行动 ⟶ 观察 ⟶ 再思考
🎯 选什么模型 = 选什么芯片，决定 Agent 能力上限

📷 Memory = 手机的相册 + 备忘录

存储对话记忆和知识，让 AI「记住」你

📱 手机上

📸 相册自动按时间、地点、人脸分类
📝 备忘录存你的笔记和想法
🔍 搜索框一搜就能找到旧照片

✿ OpenClaw 里

🗃 SQLite 存储所有对话历史
🔭 BM25 + 向量混合搜索找到相关记忆
⏲ 时间衰减：越久远的记忆权重越低（像你很少翻旧照片）

📦 Skills = 手机的应用商店

想让 AI 干更多事？装个 Skill 就行，就像装 App

📱 手机上

🎭 想打游戏？装个游戏 App
📷 想修图？装个修图 App
🛒 App Store 有几百万个 App 可选

✿ OpenClaw 里

🔧 底层 Tools: 搜索、读文件、调 API（像系统自带 App）
🚀 高层 Workflows: 组合多个 Tool 的复杂流程（像快捷指令）
🏠 ClawHub 市场: 社区共享的 Skills（像 App Store）

⏰ Heartbeat = 手机的闹钟 + 后台刷新

不用你喊它，它自己会定时做事

📱 手机上

⏰ 闹钟到点就响，不用你盯着
🔄 后台刷新：微信自动收新消息
📅 日历提醒：到时间就弹通知

✿ OpenClaw 里

🕐 定时任务：每天 9 点自动发日报
👀 主动行为：检测到异常主动告警
💋 心跳检测：确认 Agent 还活着、还在工作
📱 手机没有操作系统 = 一块砖 ⟶ 🧠 大模型没有 OpenClaw = 只会聊天的 AI
📱 手机 🔔
通知中心
你手机上的所有消息——微信、短信、钉钉、外卖——都先到通知中心，然后再分发到各个 App。
💬
微信 老板: 今天报告写好了吗？
📧
邮件 会议邀请: 下午 3 点评审
🕵
外卖 您的午餐已送达一楼
💰
银行 工资已到账 ￥15,000.00
✿ OpenClaw 📨
Gateway 网关
所有渠道的请求——飞书消息、Slack 指令、API 调用、定时触发——都先到 Gateway，然后路由到对应的 Agent。
  • 🔬
    消息路由 根据消息来源和内容，分发给正确的 Agent 处理
  • 📋
    Lane Queue 排队 多个请求同时到达？按优先级排队，不会互相冲突
  • 🛡
    协议转换 不同来源用不同格式？Gateway 统一翻译成 Agent 能理解的语言
🔔 通知中心：统一收取，分发到各 App ⟶ 📨 Gateway：统一接收，路由到各 Agent
📱 手机 ⚡
处理器 (CPU)
手机的大脑。你点外卖、刷视频、拍照片，所有操作最终都是 CPU 在运算。芯片越好，手机越快越聪明。
  • 💫
    运行所有 App 的逻辑 每次点击按钮，CPU 都在背后飞速计算
  • 🔄
    多任务切换 边听歌边回消息，CPU 在几个任务间快速切换
  • 📈
    性能决定体验 A17 Pro 跑得动原神，低端芯片只能看看微信
✿ OpenClaw 🧠
Brain 大脑
Agent 的思考引擎。调用大模型来「想」，然后决定下一步做什么。选什么模型 = 选什么芯片。
  • 💡
    调用大模型推理 把问题发给 GPT/Claude，拿回思考结果
  • 🔁
    Agentic Loop 循环 思考 ⟶ 行动 ⟶ 观察结果 ⟶ 再思考... 直到任务完成
  • 🎯
    模型 = 芯片 GPT-4o 是旗舰芯片，GPT-4o-mini 是中端芯片——按需选择
⚡ CPU：所有操作的执行引擎 ⟶ 🧠 Brain：所有思考的推理引擎
📱 手机 📷
相册 + 备忘录
手机帮你记住一切：照片按时间线排列，备忘录存你的想法，搜索框帮你快速找到想要的内容。
  • 📅
    按时间线组织 今天拍的照片在最前面，去年的往后排
  • 🔍
    智能搜索 搜「猫」，自动找出所有有猫的照片
  • 🗑
    自动清理 存储满了？建议删除模糊照片和截图
✿ OpenClaw 🗃
Memory 记忆系统
让 Agent 能「记住」以前聊过什么。存储对话历史和知识，需要时精准检索出来。
  • 🗃
    SQLite 持久存储 所有对话都存下来，重启也不会丢
  • 🔭
    混合检索 BM25（关键词匹配）+ 向量搜索（语义理解）双管齐下
  • ⏲
    时间衰减 越久远的记忆权重越低——就像你很少翻三年前的照片
📷 相册/备忘录：存储、分类、搜索你的记忆 ⟶ 🗃 Memory：存储、检索、衰减 Agent 的记忆
📱 手机 📦
应用商店 (App Store)
新买的手机只有基础功能，装了 App 才能点外卖、打车、刷视频。App Store 让手机能力无限扩展。
  • 💻
    系统自带 App 电话、短信、相机——出厂就有，删不掉
  • 🛒
    第三方 App 微信、抖音、美团——按需下载安装
  • 🛠
    快捷指令 组合多个 App 的操作，一键自动执行
✿ OpenClaw 🔧
Skills / Tools 技能系统
Agent 自带基础工具，还能从 ClawHub 市场安装更多技能。两层架构：底层 Tools + 高层 Workflows。
  • 🔧
    底层 Tools（系统自带 App） 搜索、读文件、调 API、执行代码——基础能力
  • 🚀
    高层 Workflows（快捷指令） 组合多个 Tool 的复杂流程，比如「分析日志 + 生成报告 + 发邮件」
  • 🏠
    ClawHub 市场（App Store） 社区共享的 Skills，一键安装扩展 Agent 能力
📦 App Store：装 App 扩展手机能力 ⟶ 🔧 ClawHub：装 Skills 扩展 Agent 能力
📱 手机 ⏰
闹钟 + 后台刷新
你不用一直盯着手机，闹钟到点会响，App 会在后台自动收消息、同步数据。
  • ⏰
    闹钟/提醒 设好时间，到点就提醒你，不需要你盯着
  • 🔄
    后台刷新 锁屏了微信也能收消息、邮件也能自动同步
  • 📅
    日历事件 提前 15 分钟弹窗：「3 点有会议」
✿ OpenClaw 💓
Heartbeat 心跳系统
让 Agent 不只是「被动回答」，还能「主动行动」。定时任务、健康检查、异常告警。
  • 🕐
    定时任务 (Cron) 每天 9 点自动发日报、每周一生成周报
  • 👀
    主动监控 检测到服务器异常，不等你问就主动告警
  • 💓
    心跳检测 定期检查 Agent 状态——活着？忙着？卡住了？
⏰ 闹钟/后台刷新：不用你盯着也能自动运行 ⟶ 💓 Heartbeat：定时任务 + 主动行为 + 健康监控

🔒 安全问题也一样——手机会遇到的，AI Agent 也会遇到

用你熟悉的手机安全问题，理解 AI Agent 面临的安全挑战

📩

Prompt Injection 提示注入

📱 手机上 = 钓鱼短信

✿ OpenClaw = 恶意 Prompt 劫持 Agent 行为

你收到「银行」短信让你点链接改密码——其实是骗子。Agent 也一样：用户输入里藏着恶意指令，让 Agent 做不该做的事。

💀

恶意 Skills / 供应链攻击

📱 手机上 = 假冒 App / 恶意软件

✿ OpenClaw = ClawHub 上的恶意 Skills

在非官方渠道装了假的「微信」，窃取你的聊天记录。ClawHub 上也发现过 1,467 个恶意 Skills，伪装成正常工具偷数据。

🎥

权限越权

📱 手机上 = App 偷开摄像头/麦克风

✿ OpenClaw = Agent 越权访问敏感数据

手电筒 App 要求访问通讯录？明显不合理。Agent 也可能越权——只让它查天气，它却去读你的邮件。权限管理是关键。

📸

数据泄露

📱 手机上 = 聊天记录被截图外传

✿ OpenClaw = 对话数据被发送到外部

同事截了你的聊天记录发到群里。Agent 处理的敏感数据也可能被泄露——通过 Tool 调用传到第三方服务器，或者写入不安全的日志。

⚠ 核心原则：像管理手机 App 权限一样管理 Agent 权限——最小权限、按需授权、定期审查
