Agent 之间的语言是什么？

JSON 结构化消息

任务分配、状态更新、工具调用都使用 JSON 格式，确保机器可解析。

自然语言推理

Agent 内部的思考和推理用自然语言完成，类似人类的 "内心独白"。

System Prompt 定义行为

每个 Agent 的角色、能力边界和输出格式由 System Prompt 严格定义。

Tool Calls 执行动作

Agent 通过 tool_call 调用外部工具（读写文件、运行命令等）来执行实际操作。
