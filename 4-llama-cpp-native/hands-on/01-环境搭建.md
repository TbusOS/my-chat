# 第一章：环境搭建

## 本章目标

搭建 llama.cpp 开发环境：
1. 克隆项目
2. 编译
3. 下载模型
4. 运行测试

---

## 1.1 克隆项目

```bash
git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
```

---

## 1.2 编译

### 1.2.1 Linux/macOS

```bash
mkdir build
cd build
cmake ..
make -j$(nproc)
```

### 1.2.2 macOS (Metal 加速)

```bash
cmake .. -DLLAMA_METAL=ON
make -j$(nproc)
```

### 1.2.3 CUDA 支持

```bash
cmake .. -DLLAMA_CUBLAS=ON
make -j$(nproc)
```

---

## 1.3 下载模型

```bash
# 使用 HuggingFace CLI
huggingface-cli download \
  TheBloke/Llama-2-7B-Chat-GGUF \
  llama-2-7b-chat.Q4_K_M.gguf \
  --local-dir models/
```

---

## 1.4 运行测试

```bash
# 交互式对话
./build/bin/llama-cli \
  -m models/llama-2-7b-chat.Q4_K_M.gguf \
  -n 256 \
  -p "Write a hello world in Python:"

# 启动服务器
./build/bin/llama-server \
  -m models/llama-2-7b-chat.Q4_K_M.gguf
```

---

## 1.5 常见问题

### 编译失败

```bash
# 安装依赖
apt install cmake build-essential  # Linux
xcode-select --install            # macOS
```

### 模型加载失败

```bash
# 检查模型文件
ls -lh models/
```
