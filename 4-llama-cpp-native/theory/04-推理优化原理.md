## 1.1 KV Cache

### 1.1.1 什么是 KV Cache？

在自回归生成中，保存之前计算的 K 和 V：

```
第一次: Q1 × K1,V1 → O1
第二次: Q2 × [K1,K2], [V1,V2] → O2
```

### 1.1.2 优化效果

| 方法 | 计算量 | 显存 |
|------|--------|------|
| 无 Cache | O(n²) | 低 |
| 有 Cache | O(n) | 增加 |

## 1.2 批处理

### 1.2.1 静态批处理

同时处理多个请求：

```
请求1: "Hello" →
请求2: "Hi"    → 合并 → 模型 → 输出1, 输出2
请求3: "Hey"   →
```

### 1.2.2 动态批处理

根据实际长度动态分组：

```cpp
// 动态批处理示例
struct Request {
    std::vector<int> tokens;
    int max_length;
};

std::vector<Request> queue;

// 按长度排序
std::sort(queue.begin(), queue.end(),
    [](auto& a, auto& b) { return a.max_length < b.max_length; });
```
