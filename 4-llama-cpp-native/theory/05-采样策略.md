## 1.1 Greedy 采样

### 1.1.1 原理

始终选择概率最高的 token：

```cpp
// 伪代码
int greedy_sample(const float* logits, int vocab_size) {
    int best_idx = 0;
    float best_prob = logits[0];

    for (int i = 1; i < vocab_size; i++) {
        if (logits[i] > best_prob) {
            best_prob = logits[i];
            best_idx = i;
        }
    }
    return best_idx;
}
```

### 1.1.2 特点

- 确定性输出
- 可能陷入重复
- 适合代码、数学等精确任务

## 1.2 Top-K 采样

### 1.2.1 原理

从概率最高的 K 个 token 中随机选择：

```cpp
int top_k_sample(const float* logits, int vocab_size, int k) {
    // 1. 找出 top-k
    std::vector<std::pair<float, int>> scores;
    for (int i = 0; i < vocab_size; i++) {
        scores.push_back({logits[i], i});
    }
    std::partial_sort(scores.begin(), scores.begin() + k,
                      scores.end(), std::greater<>());

    // 2. 计算概率
    float sum = 0;
    for (int i = 0; i < k; i++) {
        scores[i].first = exp(scores[i].first);
        sum += scores[i].first;
    }
    for (int i = 0; i < k; i++) {
        scores[i].first /= sum;
    }

    // 3. 加权随机选择
    float r = (float)rand() / RAND_MAX;
    float cumsum = 0;
    for (int i = 0; i < k; i++) {
        cumsum += scores[i].first;
        if (r <= cumsum) {
            return scores[i].second;
        }
    }
    return scores[k-1].second;
}
```

## 1.3 Top-P (Nucleus) 采样

### 1.3.1 原理

从累积概率达到 P 的最小集合中选择：

```
排序后的概率: [0.5, 0.3, 0.1, 0.1]
累积概率:     [0.5, 0.8, 0.9, 1.0]

P=0.9 → 选取前3个
```

### 1.3.2 代码

```cpp
int top_p_sample(const float* logits, int vocab_size, float p) {
    // 1. 排序
    std::vector<float> sorted = sort_desc(logits, vocab_size);

    // 2. 找阈值
    float cumsum = 0;
    int cutoff = 0;
    for (int i = 0; i < vocab_size; i++) {
        cumsum += sorted[i];
        if (cumsum >= p) {
            cutoff = i + 1;
            break;
        }
    }

    // 3. 在前 cutoff 个中采样
    return top_k_sample(logits, vocab_size, cutoff);
}
```

## 1.4 Temperature

### 1.4.1 原理

调整概率分布的"平滑度"：

```
soft = softmax(logits / temperature)

temperature = 0 → greedy
temperature = 1 → 原始分布
temperature > 1 → 更随机
```

### 1.4.2 代码

```cpp
void apply_temperature(float* logits, int vocab_size, float temp) {
    for (int i = 0; i < vocab_size; i++) {
        logits[i] /= temp;
    }
    softmax(logits, vocab_size);
}
```

## 1.5 组合使用

### 1.5.1 常用组合

```cpp
// 高质量生成
logits = apply_temperature(logits, 0.7);
logits = top_p_filter(logits, 0.9);
token = sample(logits);

// 创意生成
logits = apply_temperature(logits, 1.2);
logits = top_k_filter(logits, 50);
token = sample(logits);
```
