## 1.1 量化原理

### 1.1.1 什么是量化？

将高精度浮点数转换为低精度表示：

```
FP32: 3.1415926535...  (4 bytes)
INT8: 31               (1 byte)
```

### 1.1.2 量化公式

```
量化: q = round(x / scale + zero_point)
反量化: x = (q - zero_point) * scale
```

## 1.3 K-Quant

### 1.3.1 K-Quant 原理

将权重分块，每块独立量化：

```
原始: [w1, w2, w3, w4, w5, w6, w7, w8]
       ↓ 分块 (每4个)
量化: [q1, q2, scale, offset]
```

### 1.3.2 量化位数

| 方法 | 位宽 | 块大小 |
|------|------|--------|
| Q4_0 | 4 | 32 |
| Q4_K | 4 | 256 |
| Q5_0 | 5 | 32 |
| Q5_K | 5 | 256 |

## 1.5 代码实现

### 1.5.1 简单量化

```cpp
// FP32 -> INT4 量化示例
void quantize(const float* src, uint8_t* dst, int n) {
    // 找到最大值
    float max_val = 0;
    for (int i = 0; i < n; i++) {
        max_val = std::max(max_val, std::abs(src[i]));
    }

    float scale = max_val / 7.0f;  // 4bit: 0-7

    for (int i = 0; i < n; i++) {
        dst[i] = (uint8_t) std::round(src[i] / scale);
    }
}
```
