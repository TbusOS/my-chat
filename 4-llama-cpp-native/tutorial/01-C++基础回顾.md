## 1. 指针和引用

### 1.1 指针

```cpp
int a = 10;
int* p = &a;  // p 指向 a 的地址

std::cout << *p;  // 解引用，获取值
```

### 1.2 引用

```cpp
int a = 10;
int& ref = a;  // ref 是 a 的引用

ref = 20;  // 修改 ref 等于修改 a
```

## 2. 类和对象

```cpp
class Model {
private:
    float* weights;
    int size;

public:
    // 构造函数
    Model(int n) : size(n) {
        weights = new float[n];
    }

    // 析构函数
    ~Model() {
        delete[] weights;
    }

    // 方法
    void forward(float* input, float* output);
};
```

---

## 3. 模板

```cpp
// 函数模板
template<typename T>
T max(T a, T b) {
    return a > b ? a : b;
}

// 使用
int m = max(1, 2);
float f = max(1.5f, 2.5f);
```

---

## 4. 智能指针

```cpp
#include <memory>

// 自动管理内存
auto ptr = std::make_unique<float[]>(100);
// 不需要手动 delete
```

---

## 总结

- 指针：内存地址
- 引用：别名
- 模板：泛型编程
- 智能指针：自动内存管理
